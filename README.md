## 專案介紹

以 React 進行開發，功能包含新增todo、切換完成/未完成、點擊兩下編輯 todo、刪除 todo。

完成的元件包含：

Header
Main (Register & Cart)
Footer
使用指南
將專案 clone 至本地，至終端機輸入：

git clone https://github.com/Gino-Hsu/react-todolist.git
執行專案，輸入：

npm start
瀏覽器 "http://localhost:3000" ，可瀏覽此專案。

停止使用，至終端機：

ctrl + c
